Technology

  • Technology
    • Scripting Language: PHP 7.3
    • Databaseserver: MySQL 5.7
    • Webserver: Apache or PHP's built-in server
  • Recommended software: a *AMP-stack
    • WAMP on Windows: Aviatechno's WampServer
      PHP CLI*: Add C:\wamp64\bin\php\version\ to PATH via Environment Variables
    • MAMP On OSX
      PHP CLI*: Paste this on the Terminal:
      echo 'export PATH=/Applications/MAMP/bin/php/phpversion/bin/:$PATH' >> ~/.bash_profile
    • Even better: a virtual LAMP-environment with Docker
      • We have prepared such an environment (portable DB!), to be used in the Mac-lab
* the first lab session we will only use PHP CLI and no webserver

Cloning the Course Materials

  • Download and install Git
    • Choose “Run Git from the Windows Command Prompt” during install
  • Clone the repository 
    cd c:\wamp64\www
git clone https://github.com/mjoris/ws-ss-course-materials.git
    • Visit your copy at http://localhost/ws-ss-course-materials
  • Update the repository regularly to get the latest version 
    cd c:\wamp64\www\ws-ss-course-materials
git reset --hard
git pull
Note: This is an example on Windows using Wampserver

When stuck ...

Help yourself ...

  • Breathe in, breathe out, and THINK!
    • Error visible? Read it!
      • PHP will tell you what went wrong in which file and on what line
    • Correct URL? Correct variable? Does the variable contain anything?
      • Debugsession
      • use var_dump() and the like
    • TIP: Rubber Duck Debugging
  • Use the built-in help
    • Syntax Highlighting
    • Function autocompletion
    • Function documentation (PHPDoc)

... or get helped

  • Keep the 10 minute thinking rule in mind!
    You may only interrupt another co-worker to ask a question if you have sat on your own for ten minutes and tried to figure out the solution yourself. When asked, you must be able to demonstrate that you’ve made some effort.
    • Check the slides and code examples
    • Check http://php.net/nameofthefunction
    • Check Google (#lmgtfy)
  • Ask a fellow student or your lecturer
